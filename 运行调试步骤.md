# EKANBAN 看板程序运行调试指南

由于该程序是基于 **React**, **Tailwind CSS**, **Framer Motion** 和 **Lucide React** 开发的，你需要一个前端开发环境来运行它。推荐使用 **Vite** 快速搭建。

## 1. 环境准备
确保你的电脑上已安装 [Node.js](https://nodejs.org/) (推荐 LTS 版本)。

## 2. 快速创建项目
在终端（Terminal 或 PowerShell）中执行以下命令：

```bash
# 创建一个新的 Vite 项目 (选择 React + TypeScript)
npm create vite@latest ekanban-app -- --template react-ts

# 进入项目目录
cd ekanban-app

# 安装核心依赖
npm install

# 安装看板需要的额外库
npm install lucide-react framer-motion tailwindcss postcss autoprefixer
```

## 3. 配置 Tailwind CSS
初始化 Tailwind 配置：
```bash
npx tailwindcss init -p
```

编辑生成的 `tailwind.config.js` 文件，修改 `content` 部分：
```javascript
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

在 `./src/index.css` 中添加 Tailwind 指令（删除原有内容）：
```css
@tailwind base;
@tailwind components;
@tailwind utilities;

body {
  margin: 0;
  background: black;
  overflow: hidden;
}
```

## 4. 部署代码
将我为你生成的 `App.tsx` 内容复制到项目的 `src/App.tsx` 中（覆盖原有内容）。

## 5. 运行预览
执行以下命令启动开发服务器：
```bash
npm run dev
```
点击终端中显示的地址（通常是 `http://localhost:5173`）即可在浏览器中查看效果。

## 6. 调试建议
- **全屏查看**：看板设计是为大屏优化的，建议在浏览器中按 `F11` 进入全屏模式查看最佳效果。
- **数据调整**：你可以直接修改 `App.tsx` 顶部的 `DATA` 常量来查看不同数据下的 UI 表现。
- **轮播控制**：鼠标悬停在页面底部可以呼出控制栏，进行手动切换或修改播放间隔。
